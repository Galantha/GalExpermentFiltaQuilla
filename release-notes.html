Version 4.0b2 First official Beta release for Thunderbird 115. FiltaQuilla had to have some necessary changes for Thunderbird 115, especially custom search terms and actions could not be edited anymore. If you find any new problems, related to the update to ESR115, please report them at FiltaQuilla issue 223 All new development for Thunderbird 115 will be published in as new branch of FiltaQuilla code, starting with a new major version number 4.*. If you would like to support FiltaQuilla with donation you can now do so here Improvements

Bug Fixes
